使用 hexo 和 github搭建个人博客

[TOC]

1.windows电脑安装git

官网下载：https://git-scm.com/downloads

2.安装Node

  1. 去node的官网下载Node.js进行安装：https://nodejs.org/en/；
  2. 安装：建议安装到非系统盘(比如这里安装到D盘)，全部默认配置，一直点击下一步。
  3. 命令行执行 node -v，如果安装成功，会有信息显示。
  4. 默认已经安装了npm (nodejs包管理器)，直接命令行执行 npm -v。
  5. 配置npm的全局模块的存放路径以及cache的路径，在NodeJS的主目录下，建立”node_global”及”node_cache”两个文件夹。

    说明：这里的环境配置主要配置的是npm安装的全局模块所在的路径，以及缓存cache的路径，这里也可以不用新建目录，如此的话，会在【C:\Users\用户名\AppData\Roaming\npm】路径中自动建立文件夹。
    由于我不希望放在C盘，可以将全模块所在路径和缓存路径放在NodeJS安装路径下，比如我这里在【D:\Program Files\nodejs】下创建两个文件夹【node_global】及【node_cache】如下图：
  6. 修改路径 考虑到墙的原因，可以使用淘宝镜像，速度更快： 完毕之后执行 cnpm -v 检验是否成功
  7. 安装hexo框架 注意，以上的命令都建议使用管理员权限。

以上两步，是环境的安装。在一台电脑上安装好之后，就基本可以不用管了。
